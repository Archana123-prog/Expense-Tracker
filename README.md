# 🪙 Expense Tracker

A simple and responsive **Expense Tracker Web App** built using **HTML, CSS, and JavaScript**.
It helps users track their income and expenses with real-time balance updates.

## 🌐 Live Demo

👉 https://archana123-prog.github.io/Expense-Tracker/

## 🚀 Features

* ➕ Add income & expenses
* 📊 Automatic balance calculation
* 📈 Income & expense summary
* 🗑️ Delete transactions
* 💾 Data saved using **LocalStorage**
* 📱 Fully responsive UI

## ⚙️ How It Works

1. Enter description and amount

   * Positive value → Income
   * Negative value → Expense

2. Click **Add**

3. Balance, income, and expense update automatically

4. Data is stored in browser (no backend required)
